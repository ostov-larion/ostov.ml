# Генеративный ascii-арт в Bash
```ascii
/  //.// . /..  ./  . / / .// //
.  // ././   . /// . ..   ./ ./
 ./ ../ ././../ ./   .//.  /// .
./ /..// //. .../  /./ /// .///
../. ./// . ./.// .... ..   .
...././  .. .. .// ./  .   //. /
.  ../.    . /     ...////...
 . // //./  ./.  / . /...// / /.
.././///// /  / /////   /.. / //
     /    ../.  .  . / /.//.  /.
/ /.//. ../.// .//  ./  //..// /
 . // .  .     ./// /..// ./.../
.  ..///..  //. .//.   .// /.. .
 /  //  /.../ .     / ///.. / .
/    // .//./ // /.///./... .. /
/./ //   /// ..  //.  / //.///./
```

Этот красивый квадратик сгенерирован одной строкой в терминале:
```bash
cat /dev/urandom | tr -dc './ ' | fold -w 32 | head -n 16
```

## Разбираемся
Работает это просто:
* `cat /dev/urandom` - получаем рандомные данные
* `tr -dc './ '` - выбираем из них только те символы, которые нам нужны.
* `fold -w32` - обрезаем строки по ширине 32 символа.
* `head -n 16` - читаем только первые 16 строк. Если не использовать эту команду, то получим в терминале красивую анимацию дождя.
